# **App Name**: Ethical Collab

## Core Features:

- Company Profile Creation: Allow companies to register and create a detailed profile with company name, domain, description, and verified email login.
- Project Posting and Management: Enable companies to post short-term projects with detailed descriptions, required skills, expected deliverables, timelines, and compensation structures, including winner and participation rewards.
- Developer Profile Creation: Enable developers/designers to create professional profiles showcasing their skills, experience level, portfolio links, GitHub/Behance/Dribbble profiles, bio, and availability.
- AI-Powered Skill Matching: Leverage AI to match developer skills with project requirements, automatically suggesting relevant projects to developers based on their profile and skills. Use an AI tool to detect the fit of the developer skills and project requirement.
- Submission Review and Scoring: Implement a scoring system for companies to evaluate submissions based on code quality, creativity, and completion, with feedback fields for providing detailed reviews.
- Fair Compensation and Reward Distribution: Automate the distribution of rewards, ensuring that all shortlisted contributors receive participation compensation and the winner receives the main reward, with clear and transparent reward logic.
- Role-Based Project Dashboard: Provide role-based dashboards for companies and developers to track project information, number of applicants, shortlisted candidates, submission status, deadlines, and final results.

## Style Guidelines:

- Primary color: HSL(210, 70%, 50%) converted to hex #3399FF, providing a trustworthy and professional feel, suitable for a collaboration platform.
- Background color: HSL(210, 20%, 95%) converted to hex #F0F8FF, a light and clean backdrop to ensure readability and focus.
- Accent color: HSL(180, 60%, 40%) converted to hex #33BDBD, offering a contrasting, fresh highlight for interactive elements and key information.
- Headline font: 'Space Grotesk' (sans-serif) for headlines and short amounts of body text; if longer text is anticipated, use this for headlines and 'Inter' for body
- Use simple, modern icons to represent different project categories, skills, and actions (e.g., submit, review, reward). 
- Employ a clean, modern, and intuitive layout with clear sections for project details, applicant lists, submission reviews, and dashboards. Use a grid-based system for consistent alignment and responsiveness.
- Incorporate subtle animations for status changes, reward distributions, and notifications to enhance user engagement and provide visual feedback.
